MIQ-PlOb-M = Planobjekt-Management


Teil 1:Anleitung für Nutzer

Nach dem Login ist in jeder Ecke des Bildschirms ein Menü eingeblendet.

Links oben:

Hier werden grundsätzliche Funktionen zur Bedienung angezeigt. Wenn sich der Mauszeiger über dem jeweiligen Icon befindet, wird ein Info-Text angezeigt.


Rechts oben:

Hier werden alle Pläne aufgelistet, welche sich der Nutzer anzeigen lassen kann.
Hinweis: diese Anzeige kann je nach Nutzer unterschiedlich aussehen.


Rechts unten:

Hier werden alle Objektlayer aufgeführt, welche sich der Nutzer anzeigen lassen kann.
Hinweis: diese Anzeige kann je nach Nutzer unterschiedlich aussehen.


Links unten:

Durch Klick auf den Pfeil nach unten werden alle Objektlayer angezeigt, welche vom Benutzer bearbeitet werden können.


Arbeiten mit dem PlOb-M

1)
Nachdem Sie sich angemeldet haben, wählen Sie im Menü rechts oben einen oder mehrere Pläne aus, welche angezeigt werden sollen.
Hinweis: Es können maximal sechs Pläne gleichzeigt angezeigt werden.

2)
Sofern Sie sich Markierungen und Eingaben aus verschiedenen Objektlayern anzeigen lassen wollen, können Sie im Menü rechts unten die jeweiligen Objektlayer auswählen.

3)
Wenn Sie neue Markierungen in einem Objektlayer setzen wollen, muss dieser im Menü links unten ausgewählt werden. Gleichzeitig wird der Objektlayer automatisch im Menü rechts unten angewählt. Grundsätzlich werden Markierungen immer in dem Objektlayer angelegt, welcher im Menü links unten ausgewählt ist. Es spielt keine Rolle, welcher Objektlayer im Menü rechts unten angezeigt wird.

4)
Sobald im Menü links unten ein Objektlayer ausgewählt wurde, erweitert sich das Menü links oben. Es wird empfohlen eine Farbe auszuwählen z.B. Blau. Ohne Farbauswahl wird automatisch die Farbe grau für Markierungen genutzt. Dies hat keine Auswirkung auf die Funktionalität, jedoch ist eine andere Farbauswahl übersichtlicher.

5)
Klicken Sie nun auf und anschließend auf die Stelle, an welche die Markierung auf dem Plan liegen soll.

6)
Durch nochmaliges klicken auf Kreis öffnet sich die Dokumentationsmaske.

7)
Um die Übersichtlichkeit bei der Dokumentation auf kleinen Endgeräten (Z.B. Handy, Tablet) zu erhöhen, können die beiden rechten Menüs ausgeblendet werden durch klick auf im Menü links oben. Ein erneutes Klicken blendet die beiden Menüs wieder ein.

Teil 2:Anleitung für Admin

Anzeigen von Kartenmaterial

Je nach Zoomstufe können Sie sich entweder Google Maps oder Open Street Map einblenden lassen. Einmal eingeblendet lässt sich die Ansicht nur zurücksetzen, indem Sie Ihr Profil neu laden.
Dazu klicken Sie auf den Button "Reload" in der Maske "Arbeitsansicht".

Einen Plan in das PlOb-M laden

Durch Klick auf den Button "Plan-Upload" erscheint die zugehörige Maske, über welche ein Plan in die das PlOb-M geladen werden kann.
Klicken Sie auf "Dateien auswählen" und wählen Sie die gewünschte Datei aus. Der Plan muss im Dateiformat ".png" vorliegen.
Bestätigen Sie Ihre Auswahl und klicken Sie anschließend auf den grünen Häkchen-Button. Der Plan wird nun hochgeladen. Dieser wird Ihnen jedoch noch nicht angezeigt.

Pläne einmessen

Damit ein Plan im PlOb-M sichtbar wird, muss ein Planlayer angelegt werden und der zugehörige Plan eingemessen werden. Zu jedem Planlayer gehört genau ein Plan. Klicken Sie auf den Button "Plänergänzung und Einmessung". Es erscheint die dazugehörige Maske "Plan Integration".
Wählen Sie im DropDown Menü den gewünschten Plan aus und vergeben Sie anschließend einen Namen für den Planlayer im Feld "Planbezeichnung".
Final bestätigen Sie Ihre Eingaben durch Klick auf den grünen Button "Einfügen". Daraufhin wird ein Planlayer angelegt und der Plan wird eingeblendet. Sie können nun die gewünschte Transparenz festlegen. Bestätigen Sie Ihre Eingabe durch Klick auf den grünen Häkchen-Button.
Erst jetzt können Sie den Plan an der gewünschten Position einmessen. Dazu wählen Sie im Bereich Arbeitsebende
den erstellten Planlayer aus. Es erscheint nun in der Maske "Plan Integration" neu der Button "Einmessen".
Wenn Sie auf den Button klicken erscheinen drei PINs, mit deren Hilfe Sie den Plan exakt postionieren können.
/
Bestätigen Sie Ihre Auswahl und klicken Sie anschließend auf den grünen Häkchen-Button.

Objektlayer einfügen

Um Objekte einfügen zu können, muss mindestens ein Objektlayer in Ihrem Profil angelegt werden. Sie können mehrere Objektlayer anlegen, um die Übersichtlichkeit zu erhöhen. Klicken Sie dazu auf den Button "Zeichnen". Es erscheint die Maske "Layer".
Nun vergeben Sie einen Namen für den Planlayer im Feld "Bezeichnung" und bestätigen anschließend Ihre Eingabe durch Klick auf "NEU". Der Objektlayer erscheint nun in der Layerliste auf der rechten Seite.
Sie können die Layerliste beliebig Ein- und Ausblenden, indem Sie auf den Button "Layerliste schliessen" klicken.

Objekte anlegen und zeichnen

In einem Objektlayer können verschiedene Objekte angelegt werden. Dies können Polygone, Kreise oder Kreispunkte sein. Wählen Sie dazu in der Maske "Arbeitsebene" den Objektlayer aus, in welchem die Objekte angelegt werden sollen. Anschließend erscheint die Optionenliste über der Maske "Arbeitsebene". Wählen Sie das gewünschte, zu zeichnende Objekt aus und klicken Sie auf der Karte bzw. Plan auf die Postion, an welcher das Objekt platziert werden soll. Ihre Eingabe wird automatisch gespeichert. Sie können das Objekt nachträglich noch ändern oder löschen.
Um die Farbe für ein Objekt zu ändern, wählen Sie zuerst in der Maske über das Dropdown Menü die gewünschte Farbe aus und positionieren anschließend das Objekt an der gewünschten Stelle.

Objekte benennen und Informationen eintragen

Blenden Sie sich den gewünschten Objektlayer ein, indem Sie in der Layerliste an entsprechender Stelle den Haken setzen. Anschließend werden die zugehörigen Objekte angezeigt. Wählen Sie das gewünschte Objekt, indem Sie darauf klicken. Es erscheint nun unten links die Informationsmaske zum ausgwählten Objekt. Sofern noch keine Informationen gespeichert wurden, erscheint der Platzhalter CID. Wenn Sie auf den Stift klicken erscheint die Maske zur Eintragung der jeweiligen Objekkinformationen.
Die Kurzbeschreibung ist ein wichtiges Merkmal des jeweiligen Objektes. Nach dieser kann gesucht bzw. gefiltert werden.

Objekte suchen

Sie können Objekte über alle Objektlayer eines Profils hinweg suchen oder detailiert nur in einem Objektlayer. Wenn Sie sich alle Objekte anzeigen lassen möchten, klicken Sie auf den Button "Suche/Filter". Wichtig ist, dass kein Objektlayer in der Arbeitsebende ausgwählt ist. Sobald Sie einen Objektlayer in der Arbeitsebene auswählen, werden nur Objekte aus dem jeweiligen Layer angezeigt und auch nur noch in diesem gesucht.
Für eine Suche nutzen Sie die Maske "Suche" und klicken nach Eingabe der Kriterien auf "Suche absenden".

Kurzbeschreibung von Objekten anzeigen lassen

Zur besseren Übersicht haben Sie die Möglichkeit, alle Kurzbeschreibungen der Objekte direkt anzeigen zu lassen. Sie müssen nicht jedes einzelne Objekt anklicken, um die Kurzbeschreibung sehen zu können. Klicken Sie dazu einfach vor dem Einblenden der Layer auf den Button "Layer-Label". Um die Kurzbeschreibungen wieder auszublenden klicken Sie einfach erneut auf den Button.

Objektliste anzeigen

Sie haben die Möglichkeit, aus der Suche heraus alle angezeigten Objekte als .csv Datei herunterzuladen und anschließend (z.B. in Excel) zu bearbeiten. Dazu filtern Sie mit Hilfe der der o.g. Möglichkeiten die gesuchten Objekte heraus und klicken anschließend in der Maske "Kurz-Doku-Liste" auf "Download (CSV)". Sie können im anschließenden Dialog den Speicherort der Datei festlegen.

Layer löschen

Sie können sowohl Objektlayer, als auch Planlayer sehr einfach löschen. Dazu wählen Sie in der Maske "Arbeitsebene" den gewünschten Layer aus und klicken anschließend auf das "X".
Bestätigen Sie den angezeigten Dialog mit "OK" und klicken Sie anschließend auf den roten Häkchen Button. Erst durch Klick auf den roten Button wird der jeweilige Layer gelöscht. Sofern Sie abbrechen möchten, reicht ein Klick auf den Button "Reload".

Karte rotieren

Wenn Sie die Karte anhand der Himmelsrichtungen ausrichten wollen, klicken Sie auf den Button Doppelpfeil und halten die Maustaste gedrückt. Bewegen Sie nun die Maus, bis Sie die gewünschte Rotation erreicht haben.
Um wieder in die ursprüngliche Ausrichtung der Karte zu wechseln, klicken Sie auf den Button "Planausrichtung".

Abmelden

Um sich aus dem PlOb-M abzumelden, klicken Sie auf den Button "Abmelden".
